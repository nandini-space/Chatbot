# LucaBot 🤖

LucaBot is a simple rule-based AI chatbot built with Python as part of an Artificial Intelligence internship project. The chatbot uses predefined responses, conditional statements, and a continuous loop to interact with users.

## Features
- Handles greetings and common questions
- Personalized user interaction
- Uses if-else logic and dictionaries
- Continuous conversation loop
- Exit commands (bye, exit, quit)
- Fallback responses for unknown inputs

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run the Python file
3. Start chatting with LucaBot!

## Sample Commands
- hello
- how are you
- what is your name
- joke
- bye
